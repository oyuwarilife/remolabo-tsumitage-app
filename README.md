# リモラボ_積み上げアプリ

リモラボの朝活・昼活・夜活・積み上げタイムの参加記録＋タスク完了数を蓄積するWebアプリ。

[![Cloudflare Pages](https://img.shields.io/badge/demo-live-brightgreen)](https://remolabo-tsumitage-app.pages.dev/)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

## 📱 デモ

**🚀 Live Demo**: https://remolabo-tsumitage-app.pages.dev/

PWA対応なので、ホーム画面に追加してアプリとして使えます。

---

## ✨ 主な機能

### 1. 参加スタンプ（4種）
- ☀️ 朝活
- 🍙 昼活
- 🌙 夜活
- ⏱️ 積み上げタイム

### 2. タスク完了記録
- ボタンを押すたびにカウント増加
- 今日のカウント ＋ 累計カウント

### 3. 週ログ表示
- 月〜日の7行で今週の記録を表示
- 過去4週間まで遡れる週ナビゲーション
- **週間目標設定**（最大5個）
- **目標達成率の可視化**（プログレスバー自動表示）
  - 朝活・昼活・夜活・積み上げ・タスク数を自動判定
  - 達成度に応じて色が変化（緑：100%達成）

### 4. ものさしパネル
- 100個を上限とした縦型バー
- 達成済み部分を5色グラデーション
- 現在地に赤いラインで「今ここ」を表示

### 5. 100個達成時のお祝い画面 🎉
- カラフルな円形プログレスバー
- リモにゃんのかわいいビジュアル
- キラキラエフェクト + 紙吹雪アニメーション
- **リモにゃん語でお祝い**（「おめでとうにゃ🎉」）
- 画像保存ボタン（Canvas APIで生成）
- SNS共有ボタン（Threads, X, LINE）

### 6. バッジ・実績システム 🏆
- **18個の実績**を収集
  - **連続記録系**：3日連続、1週間連続、2週間連続、30日連続
  - **参加回数系**：朝活10回、昼活10回、夜活10回、積み上げ10回
  - **タスク数系**：10個、50個、100個、300個、500個、1000個
  - **初回達成系**：初めての記録、初めての週間サマリー
- 統計ページで実績コレクション表示
- 未アンロックは半透明で表示

### 7. データバックアップ機能 💾
- CSVエクスポート（データをダウンロード）
- CSVインポート（データを復元）
- 週1回の自動バックアップ提案

## 🎨 デザインのポイント

- **手帳型UI**: ウィークリーダイアリーのような温かみのある見た目
- **花丸エフェクト**: 参加スタンプがONになるとCSSで手書き風の丸が出現
- **5色グラデーション**: タスクが積み上がるほど色が深くなり達成感を演出
- **リモにゃん語**: 語尾に「にゃ」が付く親しみやすいメッセージ

## 🚀 使い方

### 1. アプリを開く
https://remolabo-tsumitage-app.pages.dev/

### 2. 今日の参加スタンプをタップ
朝活・昼活・夜活・積み上げタイムのいずれかをタップ。

### 3. タスク完了ボタンを押す
「✅ タスク完了！」ボタンを押すと、カウントが増える。

### 4. 週ログで振り返り
今週の記録を確認。過去4週間まで遡れます。

### 5. 100回達成でお祝い画面 🎉
累計100, 200, 300... と達成するたびに、リモにゃんがお祝いしてくれます。

## 💾 データ管理

### データ保存
- **localStorage**（ブラウザ内に保存）
- デバイス間では引き継がれません
- ブラウザのキャッシュをクリアすると消えます

### バックアップ
定期的にデータをバックアップすることをおすすめします：

1. 画面右上の「⚙️ 設定」をタップ
2. 「📥 データをダウンロード（CSV）」をタップ
3. ファイルを保存

### 復元
1. 「⚙️ 設定」→「📤 データを復元」
2. 保存したCSVファイルを選択
3. データが復元されます
