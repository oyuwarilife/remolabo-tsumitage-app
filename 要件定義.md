# リモラボ積み上げアプリ - 要件定義

## プロジェクト概要

**目的**: リモラボの朝活・昼活・夜活・積み上げタイムの参加記録とタスク完了数を可視化するWebアプリ

**対象ユーザー**: リモラボ会員

**デプロイURL**: https://remolabo-tsumitage-app.pages.dev/

---

## 現在の主要機能

### 1. 参加スタンプ（4種）
- ☀️ 朝活
- 🍙 昼活
- 🌙 夜活
- 💪 積み上げタイム

### 2. タスク完了記録
- ボタンを押すたびにカウント増加
- 今日のカウント ＋ 累計カウント
- 連続記録バッジ（2日以上で表示）

### 3. 週ログ表示
- 月〜日の7行で今週の記録を表示
- 過去4週間まで遡れる週ナビゲーション
- **週間目標設定**（最大5個）
- **目標達成率の可視化**（プログレスバー自動表示）

### 4. ものさしパネル
- 100個を上限とした縦型バー
- 達成済み部分を5色グラデーション
- 現在地に赤いラインで「今ここ」を表示

### 5. 100個達成時のお祝い画面 🎉
- カラフルな円形プログレスバー
- リモにゃんのかわいいビジュアル
- キラキラエフェクト + 紙吹雪アニメーション
- SNS共有ボタン（Threads, X）

### 6. バッジ・実績システム 🏆
- 18個の実績を収集
- 統計ページで実績コレクション表示
- 未アンロックは半透明で表示

### 7. データバックアップ機能 💾
- CSVエクスポート（データをダウンロード）
- CSVインポート（データを復元）
- 週1回の自動バックアップ提案

### 8. 統計ページ 📊
- 総タスク数
- 最長連続記録
- 平均タスク数/日
- 総参加回数
- 今月の積み上げカレンダー
- 実績コレクション

## 技術スタック

- **フロントエンド**: HTML/CSS/JavaScript（フレームワークなし）
- **データ保存**: localStorage
- **画像生成**: Canvas API
- **PWA**: Service Worker（オフライン対応、ホーム画面追加）
- **ホスティング**: Cloudflare Pages（自動デプロイ）
- **バージョン管理**: GitHub

## データ構造

```javascript
{
    totalCount: 0,           // 累計タスク数
    todayCount: 0,           // 今日のタスク数
    lastDate: "",            // 最終更新日
    weekData: {},            // 週ログデータ
    streakDays: 0,           // 連続日数
    maxStreak: 0,            // 最長連続記録
    lastBackupDate: "",      // 最終バックアップ日
    achievements: [],        // アンロックした実績IDリスト
    weeklyGoals: {}          // 週間目標（週の開始日をキーとした辞書）
}
```

## デプロイ方法

### 1. ローカル開発
```bash
# HTTPサーバーを起動（例: Python）
python -m http.server 8000
```

### 2. Cloudflare Pages デプロイ
1. GitHub リポジトリに push
2. Cloudflare Pages が自動デプロイ
3. 本番URL: https://remolabo-tsumitage-app.pages.dev/

### 3. リモラボ環境でのデプロイ
1. GitHub リポジトリを共有（コラボレーター招待 or Organization に譲渡）
2. リモラボの Cloudflare Pages アカウントで接続
3. 自動デプロイ設定
