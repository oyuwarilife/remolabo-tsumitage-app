# リモラボ_積み上げアプリ

リモラボの朝活・昼活・夜活・積み上げタイムの参加記録＋タスク完了数を蓄積するWebアプリ。

[![GitHub Pages](https://img.shields.io/badge/demo-live-brightgreen)](https://oyuwarilife.github.io/remolabo-tsumitage-app/)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

## 📱 デモ

**🚀 Live Demo**: https://oyuwarilife.github.io/remolabo-tsumitage-app/

PWA対応なので、ホーム画面に追加してアプリとして使えます。

---

## ✨ 主な機能

### 1. 参加スタンプ（4種）
- ☀️ 朝活
- 🍙 昼活
- 🌙 夜活
- ⏱️ 積み上げタイム

### 2. タスク完了記録
- ボタンを押すたびにカウント増加
- 今日のカウント ＋ 累計カウント

### 3. 週ログ表示
- 月〜日の7行で今週の記録を表示
- 過去4週間まで遡れる週ナビゲーション

### 4. ものさしパネル
- 100個を上限とした縦型バー
- 達成済み部分を5色グラデーション
- 現在地に赤いラインで「今ここ」を表示

### 5. 100個達成時のお祝い画面 🎉
- カラフルな円形プログレスバー
- リモにゃんのかわいいビジュアル
- キラキラエフェクト + 紙吹雪アニメーション
- **リモにゃん語でお祝い**（「おめでとうにゃ🎉」）
- 画像保存ボタン（Canvas APIで生成）
- SNS共有ボタン（Threads, X, LINE）

### 6. データバックアップ機能 💾
- CSVエクスポート（データをダウンロード）
- CSVインポート（データを復元）
- 週1回の自動バックアップ提案

---

## 🎨 デザインのポイント

- **手帳型UI**: ウィークリーダイアリーのような温かみのある見た目
- **花丸エフェクト**: 参加スタンプがONになるとCSSで手書き風の丸が出現
- **5色グラデーション**: タスクが積み上がるほど色が深くなり達成感を演出
- **リモにゃん語**: 語尾に「にゃ」が付く親しみやすいメッセージ

## 🚀 使い方

### 1. アプリを開く
https://oyuwarilife.github.io/remolabo-tsumitage-app/

### 2. 今日の参加スタンプをタップ
朝活・昼活・夜活・積み上げタイムのいずれかをタップ。

### 3. タスク完了ボタンを押す
「✅ タスク完了！」ボタンを押すと、カウントが増える。

### 4. 週ログで振り返り
今週の記録を確認。過去4週間まで遡れます。

### 5. 100回達成でお祝い画面 🎉
累計100, 200, 300... と達成するたびに、リモにゃんがお祝いしてくれます。

## 💾 データ管理

### データ保存
- **localStorage**（ブラウザ内に保存）
- デバイス間では引き継がれません
- ブラウザのキャッシュをクリアすると消えます

### バックアップ
定期的にデータをバックアップすることをおすすめします：

1. 画面右上の「⚙️ 設定」をタップ
2. 「📥 データをダウンロード（CSV）」をタップ
3. ファイルを保存

### 復元
1. 「⚙️ 設定」→「📤 データを復元」
2. 保存したCSVファイルを選択
3. データが復元されます
